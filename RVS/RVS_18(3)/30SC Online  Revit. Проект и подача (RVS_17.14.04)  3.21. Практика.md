## Практика

В этом уроке вы вместе с преподавателем создадите в проекте рельеф по DWG-подложке, а затем разместите на генплане окружающую застройку и деревья.

Для создания топоповерхности используйте файл из папки `SC_DWG.zip`.

Чтобы разместить здание на рельефе, сначала создайте основание для здания, а затем спозиционируйте топологию, двигая по вертикали с помощью команды Move (Переместить).
